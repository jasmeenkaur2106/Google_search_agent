# 🛰️ Gemini Live Search Agent

An AI-powered **Live Search Assistant** built using **Python**, **Streamlit**, and the **Google Gemini API**. Unlike traditional chatbots that rely only on pre-trained knowledge, this application performs **real-time Google searches** to provide accurate, up-to-date, and verifiable information.

The application combines Google's Search Grounding technology with Gemini 2.5 Flash, allowing users to ask questions about current events, technology, business, science, sports, and more while receiving responses supported by reliable web sources.

---

# 🚀 Features

- 🌐 Real-time Google Search integration
- 🤖 AI-powered responses using Gemini 2.5 Flash
- 🔍 Google Search Grounding for factual answers
- 📚 Displays clickable source references
- 💬 Interactive chat interface
- ⚡ Fast Streamlit web application
- 🔒 Secure API key handling
- 📝 Conversation history support
- 🎯 Beginner-friendly project

# ⚙️ Installation

## 1. Clone the Repository

```bash
git clone https://github.com/yourusername/google-live-search-agent.git

cd google-live-search-agent
```

---

## 2. Create a Virtual Environment (Optional)

### Windows

```bash
python -m venv venv

venv\Scripts\activate
```

### macOS / Linux

```bash
python3 -m venv venv

source venv/bin/activate
```

---

## 3. Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 4. Run the Application

```bash
streamlit run app.py
```

# 🔑 API Key

Obtain a Gemini API key from **Google AI Studio**.

Launch the application and enter your API key in the sidebar.

The key is only used for the current session and is never stored.

---

# 💡 How It Works

1. Launch the Streamlit application.
2. Enter your Gemini API key.
3. Ask any question.
4. Gemini performs a live Google Search.
5. Search results are analyzed using Gemini 2.5 Flash.
6. The application generates a factual response.
7. Source links are displayed for verification.
